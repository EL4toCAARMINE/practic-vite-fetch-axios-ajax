# 🌐 Práctica de Consumo de APIs: Fetch, Axios & AJAX 🚀

Este proyecto es un entorno de pruebas desarrollado con **Vite** y **JavaScript puro** enfocado en un solo objetivo: dominar la comunicación asíncrona con el backend, este proyecto se realizo con la ayuda de StackBlitz. 

Aquí se pone en práctica, se analiza y se compara el consumo de APIs RESTful utilizando los tres métodos de peticiones HTTP más representativos en el ecosistema de JavaScript. Es el laboratorio perfecto para entender las diferencias de sintaxis, el manejo de respuestas y los casos de uso de cada herramienta.

## 🛠 Stack Tecnológico

* **Entorno de desarrollo:** Vite ⚡
* **Lenguaje:** JavaScript (ES6+)
* **Dependencias:** Axios

## 💡 Métodos implementados

En este repositorio encontrarás la implementación de los siguientes enfoques para la obtención de datos:

* **Fetch API:** El estándar moderno y nativo de los navegadores web basados en Promesas. Ideal para no depender de librerías externas.
* **Axios:** La librería externa más popular y robusta. Destaca por formatear automáticamente las respuestas a JSON y su excelente manejo de errores.
* **AJAX (XMLHttpRequest):** El método clásico y *old school*. Aunque hoy en día se usa menos en proyectos nuevos, conocerlo es fundamental para entender las bases de las peticiones asíncronas y cómo funcionaba la web antes de la llegada de las Promesas.
